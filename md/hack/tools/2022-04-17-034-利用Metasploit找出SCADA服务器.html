# 注册账号

首先，我们在https://www.shodan.io上注册一个账号

# 获取API Key

# 在Metasploit中找出采用罗克韦尔自动化技术的SCADA系统

msfconsole
use auxiliary/gather/shodan_search
show options
set SHODAN_APIKEY 第2步获取的API Key
set QUERY Rockwell
run
